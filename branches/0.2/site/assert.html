If some check fails, print an error message then die.

Code

 function assert(check,str) {
     if (! check)
     	barph(str)
 }

Notes

The version shown here is very simple, uses no globals, and dies the first time that any assert fails. Arnold Robbins has a more sophisticated version that does not die when the first assert fails. Rather, the whole program runs and if any of his asserts sets the global flag 'assertexit', then the program exits with a non-zero exit status.
