Synopsis

Split the contents of file into an array, divided by a seperator.

Example

 
  cat("config.txt",contents,"\n")

Code

Uses

slurp o

Demo

 
 function demoCat(   n,all) {
     n=cat("cat.wak",all)
     o(all,"cat",n)
 }

Main

 
 function cat(file,all,sep) { 
    return slurp("cat " file,all,sep) 
 }
