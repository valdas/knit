deShell

Prune characters that cause shell exection. Useful for avoiding side-effects from strings from malicious users.

Example

(KNIT programmers can exercise this example using cd quill; make eg15.)

Input... 

  S="Tim `rm -rf $$HOME/*` Menzies"
  print S " ==> " deShell(S)

Output... 

  Tim `rm -rf $HOME/*` Menzies ==> Tim rm -rf HOME/* Menzies

Code

 function deShell(str) {
     gsub(/["`\$;\|&><]/,"",str);
     return str
 }
