topOfPage

Adds tag lists top of page. 

 function topOfPage(file,files,titles,tags,sgat,     i,n,out,q) {
     n   = tags[file,0]
     out = ""
     q="\""
     if (n) {
         for(i=1;i<=tags[file,0];i++) {
             out = out sep "<a href=" q "?" tags[file,i] q ">" tags[file,i] "</a>"
             sep = ", " 
         }
         out = "<p class="q"categories"q" >categories: "out"</p>"        
     }
     return out
 }
