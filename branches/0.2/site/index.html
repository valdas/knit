Welcome to the Art of Awk

This site is, and is not, about the AWK programming language.

At first glance, it is a collection of tools and tricks for Awk (more specifically, GNU Awk a.k.a gawk). But if that is all you see here, then you have missed the art of Awk, namely:

    To expose and simplify core issues by stripping away superfluous detail.

When you code in AWK, you are thinking about the problem, not the complexities of the programming language (because there aren't any: no continuations, no inheritance, no pointers, no unions, etc). So programming in AWK is a study of the essential essence of a problem (and not trivial and tedious implementation details of over-elaborated languages).

AWK code is so short that it helps you quickly build and evolve prototypes. This gives you the experience you need to find the right design principles. And you know when you've found them- everything suddenly becomes clear and clean. So our advice is to keep AWKing till the code gets simple.

For example, smart programmers know that they repeat the same tasks to solve different problems. It make sense to build tools to support those tasks. This site is about those tools, written in Awk (of course!), with a little help from Bash and Make:

  • Domain-specific languages for better design and and analysis;
  • Writing and documentation tools using literate programming;
  • Simplier builds with library management tools;
  • Continuous testing via test-driven development;
  • Data mining tools to improve system comprehension;
  • Content management and version control for all the above.

Also, this site presents example applications that apply the above tools.

When you review this code, you might be surprised at how easy it is to support these tools. And that is the Art of Awk- understanding something well enough so you can write it just short enough to get the job done.

(P.S. Note that while our main focus is using those tools on Awk, very little of these tools are Awk-specific. In fact, we show how easy it is to apply these tools to other langauges.)
